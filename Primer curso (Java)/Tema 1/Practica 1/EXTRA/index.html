Eclipse

Es una plataforma de software compuesto por un conjunto de herramientas de programacion de codigo abierto a para desarrollar lo que el proyecto llama "Aplicaciones de Cliente Enriquecido"


-Eclipse fue desarrollado originalmente por IBM como el sucesor de su familia de herramientas para VisualAge

-Eclipse fue liberado originalmente bajo la Commom , pero despues fue re-licenciado bajo la Eclipse public license.



Caracteristicas
Perspectivas, editores y vistas
Gestion de proyectos
Depurador de codigo
Extensa coleccion de plug-ins
coloreado de codigo


Ventajas
La compilacion es a tiempo real
Tiene control de versiones
Asistente (wizard) para creacion de proyectos
Pruebas unitarias


Desventajas
Consume muchos recursos del sistema.
NetBeans

Netbeans

Es un entorno de desarrollo integrado libre, hecho principalmente para el lenguaje de programacion Java. Existe ademas un numero importante de modulos para extenderlo. NetBeans IDE es un producto libre y gratuito sin restricciones de uso.


La plataforma NetBeans permite que las aplicaciones sean desarrolladas a partir de un conjunto de componentes de software llamados modulos.



Ventajas
1. Manejo automatico de la memoria.
2. Es Gratis
3. Desarrolla aplicaciones web dinamicas.
4. Desarrollar aplicaciones de servidor para foros en linea


Desventajas
1. Requiere un interprete.
2. Algunas implementaciones y librerias pueden tener codigo rebuscado.
3. Una mala implementacion de un programa en java, puede resultar en algo muy lento.
4. Algunas herramientas tienen un costo adicional
